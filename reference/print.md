# Print an ExTera Object

Print method for `ExTera` object.

## Usage

``` r
print(n = 10L, ...)
```

## Arguments

- n:

  integer scalar, number of templates to print (default is 10L)

- ...:

  ignored

## Value

Self (invisibly)

## Examples

``` r
tera <- ExTera$new()
tera
#> ── ExTera ──
#> Template library:
```
